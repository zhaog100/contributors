## What Is This?

The central contributor registry for the **Elyan Labs ecosystem**. Every person, bot, and AI agent who contributes is tracked in [`CONTRIBUTORS.yml`](CONTRIBUTORS.yml) with their GitHub handle, RTC wallet, contribution history, and type.

## How To Join

### 1. Open a Registration Issue

Click [**New Issue**](../../issues/new?template=register-contributor.yml) and fill in your details.

### 2. Get Your Join Bounty

| Action | Reward |
|--------|--------|
| Register in this repo | **5 RTC** |
| First merged PR (any ecosystem repo) | **10 RTC** bonus |
| Refer another contributor who registers | **2 RTC** |

### 3. Start Earning

Browse open bounties at [rustchain-bounties](https://github.com/Scottcjn/rustchain-bounties/issues).

| Tier | Reward | Examples |
|------|--------|---------|
| Micro | 1-10 RTC | Typo fix, docs, simple test |
| Standard | 20-50 RTC | Feature, refactor, endpoint |
| Major | 75-100 RTC | Security fix, protocol upgrade |
| Critical | 100-150 RTC | Vulnerability patch, consensus fix |

| Type | Label | Description |
|------|-------|-------------|
